* In order to run this application, we will need to first start the back-end by running "node server.js" from the backend directory in command line.
note: in order to run this application, one will need to have mongodb and node.js installed in his machine

* after having the back-end up and running, we can open the front-end directory and open the index.html file. 
* the system on which ths node application will be tested on should have an active internet connection in order to fetch the results (from the provided api in assignment statement) and store them in the database.
